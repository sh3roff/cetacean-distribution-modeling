# 🌊 Cetacean Distribution Modeling

This repository contains the code developed for my Master's research in Environmetrics, focusing on environmental data analysis and modeling to understand cetacean distribution patterns.

---

## 🇧🇷 Summary (Resumo)

Este projeto apresenta a análise e modelagem da distribuição de cetáceos com base em variáveis ambientais e oceanográficas, utilizando técnicas estatísticas e dados espaciais.

---

## 📌 Context

Cetacean distribution is strongly influenced by environmental and oceanographic conditions. Understanding these relationships is essential for species conservation and environmental management.

---

## 🎯 Objective

The main objective of this project is to analyze and model the relationship between cetacean occurrence and environmental variables, such as:

- Sea surface temperature  
- Sea surface salinity  
- Chlorophyll-a concentration  
- Mixed layer depth  
- Bathymetry (depth)  
- Distance to coast  
- Latitude  
- Longitude
  
---

## 🗂️ Data

The dataset used in this project is not publicly available due to data privacy and usage restrictions.

However, the repository includes all scripts and workflows used for data preprocessing, analysis, and modeling, which can be adapted to similar datasets.

---

## ⚙️ Methodology

The workflow was structured into the following steps:

1. **Initial data preprocessing**  
2. **Environmental data acquisition**  
3. **Data cleaning and integration**  
4. **Statistical modeling and prediction**

It includes:

- Data cleaning and preprocessing  
- Exploratory data analysis (EDA)  
- Statistical modeling  
- Spatial analysis and prediction
